

# Tip Calculator (C# Console App)

A simple C# console program that calculates a tip based on the bill amount using predefined percentage ranges.


## 📌 Tip Rules

- Bill ≤ $10  → 10% tip  
- $10 < Bill ≤ $20 → 12% tip  
- $20 < Bill ≤ $30 → 14% tip  
- $30 < Bill ≤ $50 → 15% tip  
- Bill > $50 → 20% tip  

## ▶️ How It Works

1. The program displays tip rules.
2. The user enters the bill amount.
3. The program calculates and prints the tip amount based on the range.
